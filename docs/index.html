Fundamentos Matemáticos y Físicos

Las ecuaciones e interpretaciones físicas clave que impulsan el Aprendizaje Automático Cuántico.

1. Codificación de Estados (Angle Embedding)

Para procesar variables clásicas en hardware cuántico, debemos mapear vectores del espacio real a amplitudes del espacio de Hilbert cuántico.

|\psi(\vec{x})\rangle = \bigotimes_{i=1}^n R_x(x_i) |0\rangle^{\otimes n}

Donde $R_x(\theta) = \exp\left(-i \frac{\theta}{2} X\right)$ es una rotación de espín en la esfera de Bloch. Las características clásicas se normalizan primero en $[0, \pi]$ para cubrir el rango completo de estados ortogonales.

2. Circuitos Cuánticos Parametrizados (Ansatz)

Los parámetros ajustables $\vec{\theta}$ actúan como los pesos de la red cuántica. Cada bloque de rotación `qml.Rot` se define matemáticamente como:

R(\alpha, \beta, \gamma) = \begin{pmatrix} e^{-i(\alpha+\gamma)/2}\cos(\beta/2) & -e^{i(\alpha-\gamma)/2}\sin(\beta/2) \\ e^{-i(\alpha-\gamma)/2}\sin(\beta/2) & e^{i(\alpha+\gamma)/2}\cos(\beta/2) \end{pmatrix}

Estas compuertas parametrizadas modifican la densidad de probabilidad del estado cuántico antes de realizar mediciones para estimar expectativas.

3. Entrelazamiento e Interferencia

El entrelazamiento correlaciona múltiples qubits, permitiendo al circuito modelar relaciones no lineales complejas entre características.

U_{\text{ent}} = \prod_{i=0}^{n-2} \text{CNOT}_{i, i+1}

En el clasificador cuántico, se utiliza una topología circular para entrelazar el último qubit con el primero, logrando una conectividad periódica que incrementa la expresividad del circuito.

4. Medición Parcial y Patching

El Patch GAN mide parcialmente una línea ancila. El operador de proyección de medición parcial permite reducir la dimensionalidad cuántica:

P(x_0 \dots x_3) = \text{Tr}_{q_4} \left( |\psi\rangle \langle \psi | \otimes |1\rangle\langle 1|_{q_4} \right)

Las 16 probabilidades de estados de los 4 qubits restantes se recopilan como intensidades de píxeles, logrando una representación eficiente de imágenes clásicas.

Guía de Instalación y Configuración Local

Pasos detallados para configurar e iniciar este laboratorio cuántico en su máquina local.

1. Prerrequisitos de Software

Asegúrese de tener instalado Python 3.9 o superior en su sistema local. Se recomienda instalar el gestor de paquetes uv para una descarga ultrarrápida de dependencias.

2. Configuración del Entorno Virtual

Configure el entorno virtual de Python y actívelo en su consola:

# Crear entorno virtual
python -m venv .venv

# Activar en Windows (PowerShell)
.venv\Scripts\Activate.ps1

# Activar en Linux/macOS
source .venv/Scripts/activate

3. Instalación de Dependencias

Instale las bibliotecas cuánticas y clásicas requeridas:

pip install torch pennylane pennylane-lightning scikit-learn matplotlib jupyterlab

4. Ejecución del Laboratorio

Inicie Jupyter Lab y ejecute los notebooks interactivos:

# Iniciar servidor local
jupyter lab

# Abra quatum_classifier.ipynb o quantum_patch_gan.ipynb en el panel lateral y ejecute todas las celdas.
